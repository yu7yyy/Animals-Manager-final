Animal Shelter Manager

A simple console-based program to manage an animal shelter. The program supports adding, removing, searching, and displaying animals, as well as saving and loading animal data from a file. It also logs all actions for tracking purposes.

Features

Object-oriented design with Animal base class and specific animal types:

Dog – can fetch and bark.

Cat – can purr and meow.

Parrot – can repeat phrases and greet.

Shelter manager to handle multiple animals:

Add new animals

Remove animals by name

Search for animals by name

Display all animals

Save data to animals.json

Load data from animals.json

Logging of all actions in shelter.log.
